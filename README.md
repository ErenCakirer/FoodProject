🥦 CoreAndFood: Modern Market Yönetim Sistemi
Bu proje, ASP.NET Core 6.0 MVC mimarisi kullanılarak geliştirilmiş, modern ve dinamik bir gıda stok yönetim panelidir. Proje, hem güçlü bir backend yapısı hem de kullanıcı dostu bir arayüz sunmak üzere tasarlanmıştır.

🚀 Öne Çıkan Özellikler
Modern Dashboard: Toplam ürün, kategori ve stok bilgilerini anlık olarak gösteren renkli ve dinamik istatistik kartları.

Görsel Ürün Yönetimi: Ürünlerin resim (ikon), stok, fiyat ve kategori bazlı tam CRUD işlemlerini içeren, "Retina Ready" (yüksek çözünürlüklü) görsellerle desteklenmiş bir yapı.

Akıllı Sayfalama (Pagination): X.PagedList entegrasyonu sayesinde büyük veri setlerinde hızlı ve düzenli listeleme.

Performans Odaklı: Hafif (lightweight) kod yapısı sayesinde sayfalar arası geçişte maksimum hız.

Güvenli Erişim: Cookie tabanlı Authentication sistemi ile korunan yönetim alanı.

🛠 Kullanılan Teknolojiler
Backend: C# & ASP.NET Core 6.0 MVC

Veritabanı: MS SQL Server & Entity Framework Core (Code First)

Frontend: HTML5, CSS3, Bootstrap 5

Görselleştirme: İkon destekli ürün listeleri ve Bootstrap Dashboard bileşenleri
